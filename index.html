Supported ESC Vulnerabilities

All known Active Directory Certificate Services escalation paths

ESC1

Misconfigured Certificate Templates

Exploits templates with ENROLLEE_SUPPLIES_SUBJECT flag

ESC2

Any Purpose EKU

Certificates with Any Purpose or no EKU restrictions

ESC3

Enrollment Agent Templates

Certificate Request Agent abuse for privilege escalation

ESC4

Vulnerable ACLs

Weak permissions on certificate templates

ESC5

PKI Object ACLs

Vulnerable permissions on PKI objects

ESC6

EDITF_ATTRIBUTESUBJECTALTNAME2

CA configuration allowing SAN specification

ESC7

Vulnerable CA ACLs

Weak permissions on Certificate Authority

ESC8

NTLM Relay to AD CS

HTTP enrollment endpoints vulnerable to relay

ESC9

No Security Extension

CT_FLAG_NO_SECURITY_EXTENSION flag abuse

ESC10

Weak Certificate Mappings

Vulnerable certificate authentication mappings

ESC11

IF_ENFORCEENCRYPTICERTREQUEST

Relay attacks via RPC certificate enrollment

ESC13

Issuance Policy Abuse

OID group link vulnerabilities

Quick Start

01

Clone Repository

git clone https://github.com/Hackwarts12/ESCepcion.git
02

Install Dependencies

cd ESCepcion && pip install -r requirements.txt
03

Run ESCepcion

python escepcion.py --scan --target domain.local
